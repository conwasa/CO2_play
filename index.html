About

Readings are taken using an IR absorbtion sensor with a light path length of 1m. The sensor is in a ventilated weatherproof enclosure. Its connected to logging software running on a Windows 7 laptop. The sample interval is 10 minutes. A Python program processes the data to produce CSV files for the graphs and table, and Json for statistics. The graphs and table are produced using Google's Visualisation API and the Json processed with Handlebars. Data is stored in a Google storage bucket.
